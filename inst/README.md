# ScalarEye Institution Portal - Frontend

Institutional portal for educational institutions in Zimbabwe to manage their records, programs, and students.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- TanStack Query
- Radix UI

## Getting Started

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
3. Build for production: `npm run build`
